# E-Ink Dashboard (墨水屏复古控制台)

一个专为电子墨水屏（E-Ink）设备量身打造的个人信息看板。采用纯正的 Windows 95 复古 UI 风格，彻底摒弃现代网页的圆角与彩色元素，利用极简的黑白灰高光阴影在灰阶屏幕上实现极其锐利的显示效果。

## 核心功能

- **翻页时钟**：采用全屏幕自适应 (`vmin`) 布局，彻底移除秒级动画与冒号闪烁，从根本上杜绝墨水屏每秒局部刷新带来的残影与高耗电。
- **天气**：接入 [Open-Meteo](https://open-meteo.com/) 免密钥 API，内置纯手工绘制的全套 **黑白矢量 SVG 天气图标**，支持自动适配当地经纬度。
- **资讯**：
  - 默认接入「IT之家」科技资讯。
  - 带有设置弹窗，支持**自定义任意 RSS 订阅源**。
  - 内置时间戳强缓存穿透机制，确保每次拉取都是最新内容。
- **主题反色**：利用 CSS 变量实现极速的纯黑/纯白底色切换，配置自动保存在浏览器 `LocalStorage` 中，断电重启不丢失。
- **清屏机制**：针对墨水屏的物理特性，在标签页切换时引入了 **黑白交替全屏闪烁（Global Refresh）** 机制，清除残影。
如果你需要修改天气的默认经纬度或接口刷新频率，请打开 js/app.js 修改顶部的全局配置项
**const CONFIG = {
    LAT: 30.494791,   // 默认纬度 (示例: 武汉)
    LON: 114.304569,  // 默认经度
    WEATHER_INTERVAL: 1000 * 60 * 60, // 天气刷新频率 (默认1小时)
    QUOTE_INTERVAL: 1000 * 60 * 30    // 一言刷新频率 (默认半小时)
};
